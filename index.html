Interactive Narrative

Disco-Narrative

A framework for turning a compact story requirement into a symbolic narrative model that can be tested by a planner.

LLM Automation Symbolic Planning Reachability
Interactive Flow

Story Input

A compact story premise is used as the seed for generating characters, facts, action families, and target milestone states.

Selected Node

LLM Automation

A short story prompt is expanded into a full symbolic schema: objects, facts, actions, milestones, effects, stages, and action families.

Design Goal

Keep narrative generation open-ended while still making the generated action space testable.

Verification Role

Use planner-backed reachability as a structural check before actions are presented as playable options.

Neurosymbolic AI

Neurosymbolic Clinical Trial Matching

LLM-Driven Abduction and Logical Verification for clinical trial matching.

Interactive Flow

Patient Record

The system starts from noisy and incomplete clinical text, where relevant trial evidence may be implicit or underspecified.

Problem

Clinical evidence is messy, eligibility logic is strict.

LLMs handle noisy clinical language, while symbolic verification handles deterministic thresholds, temporal constraints, and nested eligibility logic.

Method

The framework uses abductive reasoning to infer missing or underspecified clinical facts, then checks the resulting interpretation against formal logic.

Contribution

The work shows that neurosymbolic CTM can improve accuracy, specificity, robustness, and auditability over standalone LLM baselines.

Keywords

Clinical trial matching, abductive reasoning, logical verification, auditable LLM systems.
